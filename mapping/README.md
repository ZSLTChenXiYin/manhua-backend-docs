  - 静态资源模块 >> [分类资源](../static/README.md#分类资源)

### 反馈找书
![反馈找书](./反馈找书.png "反馈找书")
- 参考文档:
  - [反馈模块](../feedback/README.md)
- 说明:
  1. 先拉取用户反馈列表，这个相当于一个简短的问题汇总
  2. 当用户需要反馈时，根据需要拉取用户反馈标签列表，标签ID将用于反馈信息的type字段
  3. 用户反馈和章节纠错分别调用不同的接口，不可混淆
